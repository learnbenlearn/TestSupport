    • loadMasterDetailFields ()
    • overwriteFieldSpecs (fieldSpecs)
      Public method to overwrite all field specifications after instantiation of DataGeneration subclass. Clears all existing specifications (whether a new specification for that field is provided or not) and replaces them with provided field specifications.
    • resetRecordChanges ()
      Fluent method to reset modifiedRecords property so that it matches records property. Changes to the existing records can then be made for purposes of update calls or mocking update operations. Only needs to be called if wanting to discard existing record changes that have already been made
    • setFieldSpec (fieldSpec)
      Public method to set a field specification after instantiation of DataGeneration subclass. If a specification already exists for the provided field, this method will overwrite that existing specification. All other existing field specifications will be preserved.
    • setFieldSpecs (fieldSpecs)
      Public method to set multiple field specifications after instantiation of DataGeneration subclass. Any specifications that already exist for provided fields will be overridden by this method. All other existing field specifications will be preserved.
